How to Logde a Grievance ?

  • If you are new user, Register to our portal or else Login.
  • After logging in using the valid credentials, user can lodge a complaint.
  • The user may track the status of the Grievance on the Dashboard.
  • Once some action is taken by the Local Authorities on the complaint, the user is notified.
